# Badging-System
A Modular badge in system that uses device to scan in a qrcode, phones allow users to create and use their badge, and the admin website allows us to monitor our devices and  leaderboard

# Tech Stack
Frontend:
- iOS/Android - Expo (React Native)
- TabletView - Swift/SwiftUI
- Admin - Next.js
Backend: Nest.js
Database: MongoDB (Firebase)
Authentication: Clerk
Hosting: Vercel
